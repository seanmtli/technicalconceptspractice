# Asset Files Required

Please add the following image files to this folder:

## Required Images

### icon.png
- Size: 1024x1024 pixels
- Format: PNG
- Purpose: App icon displayed on home screen

### splash.png
- Size: 1284x2778 pixels (or similar aspect ratio)
- Format: PNG
- Purpose: Splash screen shown during app loading
- Note: Background color is set to #6200EE in app.json

### adaptive-icon.png
- Size: 1024x1024 pixels
- Format: PNG
- Purpose: Android adaptive icon foreground image
- Note: Background color is set to #6200EE in app.json

### favicon.png
- Size: 48x48 pixels
- Format: PNG
- Purpose: Web favicon (if running as PWA)

## Quick Start

For development, you can create simple placeholder images:

1. Create a 1024x1024 purple (#6200EE) square with "DP" text for icon.png
2. Create a 1284x2778 purple image with "Data Practice" text for splash.png
3. Copy icon.png to adaptive-icon.png
4. Create a 48x48 version for favicon.png

Or use online tools like:
- https://www.canva.com
- https://www.figma.com
- https://placeholderlogo.com
